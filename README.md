Setup and Running the Project

Clone the Repository (if applicable):
git clone <repository-url>
cd solar_farm_monitoring
Create a Virtual Environment (recommended):
python3 -m venv venv
Activate the Virtual Environment:
On macOS/Linux:
source venv/bin/activate
On Windows:
venv\Scripts\activate
Install the Required Dependencies:
Make sure your requirements.txt file contains necessary packages (e.g., Flask, OpenCV, NumPy). Then run:
pip install -r requirements.txt
Run the Application:
To start the Flask development server, execute:
python app.py
The application will run at http://127.0.0.1:5000.
Open the Application in Your Browser:
Navigate to http://127.0.0.1:5000 to view the dashboard, upload a thermal image, and see the fault detection results along with the analysis metrics.
